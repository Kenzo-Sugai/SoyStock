Gerenciar Carteira de Ação

UMLUseCase
SoyStock :: Diagrama de Caso de Uso :: Gerenciar Carteira de Ação

Description

Ator: Cliente

Fluxo Principal:

  1. O cliente faz login na plataforma do SoyStock.
  2. Na área de carteira, o cliente visualiza a lista de ações que possui em sua carteira.
  3. O cliente pode consultar informações sobre cada ação, como preço atual, valor total da posição, entre outros.
  4. O cliente pode selecionar a opção de "Gerenciar Carteira" para adicionar, remover ou modificar ações em sua carteira.
  5. O sistema valida as operações solicitadas pelo cliente.
  6. Após a validação bem-sucedida, o sistema atualiza a carteira do cliente.
