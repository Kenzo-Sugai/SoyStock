Depositar e sacar dinheiro

UMLUseCase
SoyStock :: Diagrama de Caso de Uso :: Depositar e sacar dinheiro

Description

Atores: Cliente e Banco

Fluxo Principal:

  1. O cliente faz login na plataforma do SoyStock.
  2. Na área de conta ou carteira, o cliente seleciona a opção de "Depositar" ou "Sacar" dinheiro.
  3. O sistema exibe as opções disponíveis para depósito ou saque, como transferência bancária, boleto bancário, entre outros.
  4. O cliente seleciona a opção desejada.
  5. Para depósito, o cliente fornece os dados necessários, como valor a ser depositado e informações bancárias.
  6. Para saque, o cliente fornece o valor a ser sacado.
  7. O sistema valida as informações fornecidas pelo cliente.
  8. Após a validação bem-sucedida, o sistema realiza a operação de depósito ou saque.
  9. O cliente recebe uma confirmação da transação.
