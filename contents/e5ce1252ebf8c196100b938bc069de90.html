Emitir e alterar OC

UMLUseCase
SoyStock :: CasoDeUso :: Emitir e alterar OC

Description

Atores: Cliente e B3

Fluxo Principal:

  1. O cliente faz login na plataforma do SoyStock.
  2. Na tela de negociação, o cliente seleciona a opção de "Emitir Ordem de Compra."
  3. O sistema exibe um formulário para preenchimento da ordem de compra, incluindo informações como ação desejada, quantidade, preço, entre outros.
  4. O cliente preenche o formulário com os dados da ordem de compra.
  5. O sistema valida as informações fornecidas pelo cliente.
  6. Após a validação bem-sucedida, o cliente confirma a ordem de compra.
  7. O sistema envia a ordem para a Bolsa de Valores (B3) através das APIs de integração.
  8. O cliente pode acompanhar o status da ordem de compra.
