Gerenciar OC ou OV em andamento

UMLUseCase
SoyStock :: CasoDeUso :: Gerenciar OC ou OV em andamento

Description

Atores: Cliente e B3

Fluxo Principal:

  1. O cliente faz login na plataforma do SoyStock.
  2. Na área de negociação, o cliente pode acessar a lista de ordens de compra e venda em andamento.
  3. O cliente pode selecionar uma ordem em andamento para visualizar detalhes e opções de modificação ou cancelamento.
  4. O sistema valida as operações solicitadas pelo cliente.
  5. Após a validação bem-sucedida, o sistema permite ao cliente modificar a ordem ou cancelá-la.
  6. O cliente recebe confirmação das ações realizadas.

Fluxo Alternativo:

5.1 - O cliente não consegue modificar ou cancelar a ordem por falha do sistema 5.2 - O cliente acessa a aba "Precisa de Ajuda?" 5.3 - O cliente entra na área de interesse 5.4 - Envia um formulário com seu problema 5.5 - O sistema recebe e envia para a equipe de Atendimento Interno 5.6 - O problema é resolvido e retorna ao passo 6.
