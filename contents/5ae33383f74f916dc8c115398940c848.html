Cadastro do Usuário

UMLUseCase
SoyStock :: Diagrama de Caso de Uso :: Cadastro do Usuário

Description

Ator: Cliente

Fluxo Principal:

  1. O cliente acessa a plataforma do SoyStock.
  2. Na tela inicial, o cliente escolhe a opção de "Cadastro de Usuário".
  3. O sistema exibe um formulário de cadastro, solicitando informações pessoais, como nome, CPF, data de nascimento, endereço, e-mail, telefone, entre outros.
  4. O cliente preenche o formulário com os dados corretos.
  5. O sistema valida os dados fornecidos.
  6. Após a validação bem-sucedida, o sistema cria uma conta para o cliente e envia um e-mail de confirmação.
  7. O cliente recebe o e-mail e segue as instruções para confirmar o cadastro.
  8. Após a confirmação, o cliente pode fazer o login no sistema.
