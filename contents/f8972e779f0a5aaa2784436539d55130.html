Emitir e alterar OV

UMLUseCase
SoyStock :: CasoDeUso :: Emitir e alterar OV

Description

Atores: Cliente e B3

Fluxo Principal:

  1. O cliente faz login na plataforma do SoyStock.
  2. Na tela de negociação, o cliente seleciona a opção de "Emitir Ordem de Venda."
  3. O sistema exibe um formulário para preenchimento da ordem de venda, incluindo informações como ação disponível na carteira, quantidade, preço, entre outros.
  4. O cliente preenche o formulário com os dados da ordem de venda.
  5. O sistema valida as informações fornecidas pelo cliente.
  6. Após a validação bem-sucedida, o cliente confirma a ordem de venda.
  7. O sistema envia a ordem para a Bolsa de Valores (B3) através das APIs de integração.
  8. O cliente pode acompanhar o status da ordem de venda.

Fluxo Alternativo:

4.1 - O cliente incluiu uma informação errada, fazendo com que o sistema invalide as operações. 4.2 - O sistema exibe uma mnesagem de erro 4.3 - O cliente arruma os campos errados 4.4 - Avança para o passo 5.
